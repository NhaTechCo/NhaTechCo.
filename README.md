# NhaTechCo.

Website dich vu thiet ke website, ung dung di dong va phan mem cho doanh nghiep.

## Cau truc

- `frontend`: React + TypeScript + Next.js, landing page va form lien he.
- `backend`: Next.js API Routes / Server Actions, endpoint nhan lead, luu lead va gui thong bao Slack webhook khi cau hinh.

## Chay local

Mo hai terminal rieng:

```powershell
cd .\frontend
npm install
npm run dev
```

```powershell
cd .\backend
npm install
npm run dev
```

Frontend: `http://localhost:3000`

Backend: `http://localhost:4000`

API lead: `http://localhost:4000/api/lead`
